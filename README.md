# 🔐 Password Generator (Tkinter)

This is a simple and user-friendly password generator built using **Python** and **Tkinter**.  
You can generate strong passwords of different lengths and strengths with a clean graphical interface.

---

## ✅ Features

- Choose password length (4 to 32 characters)
- Select password strength: Low, Medium, Strong
- Instantly generate a random password
- Copy the generated password to clipboard
- Clean GUI interface with background image

---
